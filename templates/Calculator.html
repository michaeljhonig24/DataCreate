Welcome to the Calculator App! This very simple design allows you to
compute basic mathematical equations with an online digital calculator.

  • This calculator only computes two coeffecients at a time.
  • The data of the past several calculations will be stored on
    the side in a table. This table will update after each calculation.
  • Along with the mouse, the keyboard keys also work to input values.
  • certain keys such as "x" or "*" have the same usage in the calculator.
