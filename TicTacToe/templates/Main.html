About This Project

Overview

This web application features classic board games with AI opponents powered by machine learning and game theory algorithms. Each game implements multiple difficulty levels using reinforcement learning (Q-Learning) and optimal play algorithms (Minimax).

Technologies Used

Backend: Python, Flask, PostgreSQL
Frontend: JavaScript, HTML/CSS
AI/ML: Q-Learning, Minimax with Alpha-Beta Pruning, MCTS (Monte Carlo Tree Search), DQN (Deep Q-Network), Reinforcement Learning
