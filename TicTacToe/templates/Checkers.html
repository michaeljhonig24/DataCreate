Difficulty levels

Easy

Shallow depth bot applies a minimum depth minimax algorithm. The algorithm only looks one move ahead per turn, effectively picking the current best move, but also only accounts for the current board. Along with this, there is a 30% chance the bot moves randomly as well, making a very simple opponent.

Medium

Medium bot learns from a Deep Q Network (DQN) trained through 10000+ simulated games. These games consisted of a combination of minimax opponents and random exploration of moves in order to replicate the playstyle of an average player.

Hard

Deep searching bot uses minimax algorithm similar to the easy bot, with the caveat that it looks 5 moves ahead instead of just 1, making it pick a much more strategic move. This bot knows many possible outcomes they you might play, so beating it is not easy.
